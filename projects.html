Project Details

GitHire

GitHired is a web application that helps companies discover developers whose Github experience matches the exact skill set of an open role. Github repositories are often overlooked by recruiters, but by automatically parsing public repositories, commit history, and language usage, GitHired surfaces candidates based on what they’ve actually built, not just what they list on a resume.

Hyperloop Computer Vision Sensor System

Working with the Queen's Hyerploop Design Team, this project involved building a computer vision safety system able to detect hazards infront the hyperloop pods track using object detection. The system was then designed to output its detected object to a web interface for the user to view.

Home Environment Monitor

Designed a home enviroment monitor able to detect indoor environmental issues such as poor air quality or humidity changes using sensors. The system aims to proactively alert homeowners to potential hazards by displaying warnings, helping maintain healthier and safer living spaces.

RoboMaster CV Model

Developed and trained a computer vision model for the Queen's Knights Robotics Team to identify and track opponent robots based on LED armor plate colors in the RoboMaster competition. Our research for this project was presented at CUCAI, the largest undergraduate AI conference in Canada.
