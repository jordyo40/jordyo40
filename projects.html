Project Details

Hyperloop Computer Vision Sensor System

Built an object detection system as part of the Queen's Hyperloop Design Team, aimed at detecting obstacles and hazards on the pod's track using deep learning and computer vision techniques. Integrated model with real-time feedback loops to ensure proactive safety monitoring.

Home Environment Monitor

Created a sensor-driven environmental monitoring system to detect changes in humidity, air quality, and temperature. Displayed warnings via an LED interface and web dashboard for home health and safety.

Computer Vision for RoboMasters

Developed an AI-powered model to detect and track opponent robots during matches by identifying LED armor color patterns. Used for strategic positioning and real-time data collection in the RoboMaster competition.
